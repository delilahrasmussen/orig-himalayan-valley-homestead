## Color Palette

- **Forest Green:** Primary brand color (headings, buttons, accents)
- **Cream/Beige:** Background color (warm, inviting)
- **Stone Gray:** Text color (readable, professional)
- **Sage Green:** Accent sections
- **Terracotta:** Review stars and highlights
- **White:** Cards, contrast sections

---

## Typography

- **Headings:** Serif font (elegant, traditional)
- **Body Text:** Sans-serif font (clean, readable)
- Font sizes: Responsive scaling from mobile to desktop

## Key Features

1. **Fully Responsive Design:** Works on mobile, tablet, and desktop
2. **Image Galleries:** Lightbox viewing with navigation
3. **Interactive Elements:** Hover effects, smooth transitions
4. **Booking Integration:** Multiple booking options (direct and platforms)
5. **Contact Forms:** Easy enquiry submission
6. **Video Content:** Welcome video with audio controls
7. **Map Integration:** Google Maps for location
8. **Social Proof:** Reviews and ratings integration
